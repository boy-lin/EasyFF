# EasyFF 产品介绍

## 1. EasyFF 是什么

EasyFF 是一个桌面端 FFmpeg 图形化工具。它把常见的命令行媒体处理能力封装成可视化操作，目标是让用户在不记忆复杂参数的前提下，快速完成音视频处理任务。

一句话定位：**更易上手的 FFmpeg 桌面工作台**。

## 2. 核心价值

- 降低门槛：轻松安装FFmpeg，管理FFmpeg版本，用界面代替大量命令记忆。
- 保留灵活性：仍可直接编辑命令文本。
- 提升效率：收藏常用命令、复用历史任务、统一管理运行状态。
- 稳定可用：内置 FFmpeg 版本管理与运行时状态提示。

## 3. 主要功能

### 3.1 首页（命令执行工作台）

- 编辑/粘贴 FFmpeg 命令并执行。
- 选择输入文件与输出目录，自动辅助拼接命令。
- 实时日志查看（虚拟列表，适合长日志）。
- 输出文件完成后可一键定位到文件夹。
- 可将当前命令保存为收藏。

### 3.2 FFmpeg 版本管理

- 查看远程可用版本与本地已安装版本。
- 支持下载、取消下载、激活、删除。
- 区分“已下载 / 下载中 / 已激活 / 失败”等状态。
- 在首页执行前检查运行时，避免环境未就绪导致失败。

### 3.3 任务历史

- 查看历史任务（状态、开始时间、时长、格式等）。
- 支持分页、排序、关键词检索。
- 对已完成任务可直接打开输出目录。

### 3.4 收藏命令

- 收藏常用命令（标题、描述、命令文本）。
- 支持搜索、排序、分页。
- 支持删除与快速回填到首页继续执行。
- 支持手动同步与同步状态提示。

### 3.5 其他能力

- 多语言（i18n）支持。
- 主题模式（浅色/深色）与统一设计系统样式。
- 应用更新与强更页面支持（按配置生效）。

## 4. 典型使用流程

1. 进入首页，选择输入文件与输出目录。
2. 调整或粘贴命令并执行，观察实时日志。
3. 成功后在任务历史查看结果，必要时打开输出目录。
4. 将高频命令保存到收藏，后续一键复用。
5. 如遇运行异常，先到“FFmpeg 版本管理”检查并切换可用版本。

## 5. 适用人群与场景

- 需要频繁做转码、抽音、压缩等操作的内容创作者。
- 希望在 GUI 中使用 FFmpeg 能力的运营/测试/开发同学。
- 不熟悉命令行、但又希望保留进阶能力的普通用户。

## 6. 技术栈与架构概览

- 前端：React + TypeScript + Vite + Tailwind CSS
- 桌面壳：Tauri
- 状态管理：Zustand
- 表格与交互：TanStack Table、Radix UI
- 国际化：i18next + react-i18next

## 7. 页面入口（当前版本）

- `/`：首页（命令执行）
- `/tasks`：任务历史
- `/favorite/commands`：收藏命令
- `/ffmpeg/version-manager`：FFmpeg 版本管理
